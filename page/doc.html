Quick Static Website Builder ?

Quick Static WebSite Builder ou QSWSB est un outil permettant la création simple et automatique d'un site web. Cet outil est utilisé principalement pour la création de site web statique (sans base de données) comme des blogs, des portfolios, des catalogues ou des sites de documentation. Ses principales fonctionnalités sont:

  • La création et la gestion de composants personnalisables
  • La création visuelle de chaque pages du site avec les composants disponibles
  • La génération automatique du site web statique
  • La possibioslté de partager les composants et les sites (templates) via du JSON

QSWSB est un projet open source, vous pouvez le retrouver sur ce lien.

Les étapes de création

0 - Configuration de l'application

Vous devez l'executer sur pc, sur un serveur web et dans un navigateur supportant IndexedDB

1 - Créer des composants

Vous pouvez soit créer à partir de zéro soit vous basez sur le code html d'un composant vu sur le net. Vous pouvez aussi importer un ensemble de composants sous forme de json créés par un utilisateur qui a sauvegardé ses composants auparavant ou créés par une IA en se basant sur la structure d'un composant dans le fichier "components-template.txt" à la racine de l'application.

2 - Créer un site

Vous pouvez créer un site à partir de zéro ou en dupliquant un modèle de site (template) pour créer un nouveau site. Vous pouvez aussi importer un site sous forme de json. Vous pouvez mettre toutes les ressources nécessaires (image, données json, font, ...) dans dossier "ressource" à la racine de l'application, elles seront automatiquement incluses dans le site à condition de les appeler en précedent le chemin du fichier par "/ressource/".

3 - Builder le site

Après avoir finir la création de votre site, sauvegardez et buildez et c'est fait ! votre site statique est prêt à l'emploie, il n'y'a plus rien à toucher dans le build.

Sous le capot

Cette application est executée totalement coté client. Aucune dépendances externes. Comme base de données, elle utilise indexedDB (dexieDB) et le sessionStorage pour stocker les données des composants et des sites. Elle utilise bootstrap pour l'interface et AOS (Animation On Scroll) par defaut (expérimentale) pour les animations de vos sites. Le formatage des codes est fait par la librairie "Beautify" et le build par les libraires "JSZip" et "FileSaver". L'application est écrite en JavaScript pur (ES6) et HTML5.

Note

QSWSB a été créer dans le cadre d'allier la codage classique comme on pourrait le faire sur VSCODE avec la conception visuelle de site web. Tout sites statiques que vous pouvez faire avec un éditeur de code classique vous pouvez le faire avec QSWSB avec en plus la possibilité de créer vos pages visuellement avec les composants de votre choix.
