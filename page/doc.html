Quick Static Website Builder ?

Quick Static WebSite Builder (QSWSB) est un outil de création de sites web statiques utilisable directement dans le navigateur. Pas d'installation complexe, pas de dépendances lourdes.

Composants

Créez des éléments réutilisables avec paramètres personnalisables

Visuel

Assemblez vos pages par glisser-déposer dans l'arbre hiérarchique

Build automatique

Générez un site statique complet en un clic

Partage JSON

Exportez/importez composants et sites pour les partager

Démarrage rapide

  1. Lancez l'application
    npm start puis ouvrez http://127.0.0.1:3000
  2. Créez des composants
    Définissez du HTML avec des parties personnalisables entre {* *}
  3. Créez un site
    Cliquez sur "Créer un site", ajoutez des pages, assemblez les composants
  4. Buildez
    Cliquez sur "Builder" pour générer le site statique dans project/

Composants

Les composants sont les briques de base de vos sites. Un composant est défini par :

  • HTML : La structure du composant
  • CSS : Les styles (règles seulement)
  • JS : Le comportement (la variable component représente l'élément)
  • Paramètres : Parties personnalisables entre {* *}
  • Librairies : Dépendances CSS/JS
Les noms de paramètres doivent contenir uniquement des lettres (pas de chiffres ni symboles).

Site & Pages

Un site est composé de pages, chacune pouvant inclure :

  • Des composants organisés en arbre hiérarchique
  • Du CSS/JS global partagé entre toutes les pages
  • Du CSS/JS local spécifique à la page
  • Des librairies externes

L'éditeur de site offre une vue en arbre pour naviguer et organiser les éléments, et un aperçu visuel dans l'iframe.

Pensez à sauvegarder votre site régulièrement !

Éditeur de code

Les zones de texte contenant du code (HTML, CSS, JavaScript) utilisent Monaco Editor (le même éditeur que VS Code) pour offrir :

  • Coloration syntaxique automatique
  • Auto-complétion
  • Indentation automatique
  • Thème sombre
  • Numéros de ligne
  • Repli de code

Cliquez sur le bouton pour ouvrir l'éditeur en plein écran.

Librairies

Les librairies sont des fichiers CSS ou JS externes que vous pouvez ajouter à vos projets :

  • Fichier : Importez un fichier .css ou .js depuis votre ordinateur
  • Lien : Utilisez une URL CDN (ex: Bootstrap, jQuery)

Les librairies peuvent être incluses au niveau du site (toutes les pages) ou d'une page spécifique.

Build & Export

Deux options pour générer votre site :

Build serveur

Le site est généré dans le dossier project/ sur le serveur. Chaque projet possède son propre dossier ressources/ pour ses médias.

Export ZIP (expérimental)

Téléchargez le site complet sous forme d'archive ZIP.

Notes

QSWSB a été créé pour allier le codage classique (comme sur VS Code) avec la conception visuelle de sites web. Tout site statique que vous pouvez faire avec un éditeur de code classique, vous pouvez le faire avec QSWSB, avec en plus la possibilité de créer vos pages visuellement avec des composants réutilisables.

Ce projet est open source et disponible sur GitHub. Les retours et contributions sont les bienvenus !
